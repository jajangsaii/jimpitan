# Jimpitan Digital PWA

Aplikasi Progressive Web App untuk manajemen dana jimpitan desa secara digital dan real-time.

## 🚀 Fitur Utama

- **Multi-role System**: Admin, Petugas Lapangan, dan Warga
- **Real-time Updates**: Sinkronisasi data via Firebase Firestore
- **PWA Support**: Dapat diinstal sebagai aplikasi di smartphone
- **Offline Support**: Service Worker untuk akses offline
- **Responsive Design**: Dioptimalkan untuk mobile dan desktop

## 📋 Roles

### Admin
- Kelola data warga
- Kelola petugas lapangan
- Lihat laporan transaksi global
- Konfigurasi sistem

### Petugas Lapangan
- Input data jimpitan di lapangan
- Lihat riwayat setoran harian

### Warga
- Lihat riwayat setoran pribadi
- Lihat total kontribusi
